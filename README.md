# Calories Burned App 🔥

A Data Science mini project that predicts the number of calories burned during exercise using demographic and workout-related features. The project aims to compare multiple predictive models and identify the most accurate approach for estimating calorie expenditure.

## ✨ Features

This project covers the complete end-to-end Data Science workflow, including:

* 📊 Data preprocessing and cleaning
* 📈 Exploratory Data Analysis (EDA)
* 🤖 Machine Learning model development
* 🧠 Artificial Neural Network (ANN) development using TensorFlow/Keras
* 📏 Model evaluation and comparison
* 🚀 Streamlit deployment for real-time prediction

## 🛠️ Project Highlights

* Comparison of multiple Machine Learning regression algorithms
* Data preprocessing, encoding, and feature scaling
* Model evaluation using:

  * Mean Absolute Error (MAE)
  * Root Mean Squared Error (RMSE)
  * Mean Absolute Percentage Error (MAPE)
  * R² Score
* Hyperparameter tuning for model optimization
* Artificial Neural Network (ANN) implementation with TensorFlow/Keras
* Interactive web application built with Streamlit

## 💻 Run Locally

Clone this repository:

```bash
git clone https://github.com/yourusername/calories-burned-app.git
```

Move into the project directory:

```bash
cd calories-burned-app
```

Install all required dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```

The application will be available at:

```text
http://localhost:8501
```
